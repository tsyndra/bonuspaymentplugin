# iiko-reporter - Интеграция скриншотов таблицы

## Новая функциональность

В проект iiko-reporter добавлена возможность автоматической отправки скриншотов таблицы филиалов с сайта callcenter.hatimaki.ru.

## Расписание отправки

Скриншоты таблицы автоматически отправляются в новый чат в следующие часы по МСК:
- **12:00** - отправка отчета + скриншот таблицы
- **17:00** - отправка отчета + скриншот таблицы  
- **18:00** - только скриншот таблицы
- **19:00** - только скриншот таблицы
- **20:00** - только скриншот таблицы
- **21:00** - только скриншот таблицы

## Файлы

### Новые файлы
- `table_screenshot.py` - модуль для создания скриншотов таблицы
- `test_table_screenshot.py` - тестовый скрипт для проверки отправки

### Обновленные файлы
- `main.py` - добавлен метод `send_table_screenshot()` и логика планировщика
- `cron_report.py` - добавлена поддержка отправки скриншотов
- `requirements.txt` - добавлен playwright

## Зависимости

Новая зависимость:
```
playwright==1.54.0
```

## Установка

1. Установите playwright:
```bash
pip install playwright
playwright install chromium
```

2. Убедитесь, что все зависимости установлены:
```bash
pip install -r requirements.txt
```

## Тестирование

Для тестирования отправки скриншота таблицы:

```bash
python test_table_screenshot.py
```

## Логика работы

1. **Авторизация** - скрипт автоматически авторизуется на сайте callcenter.hatimaki.ru
2. **Поиск таблиц** - находит все таблицы с классом `.terminals-column-header`
3. **Обрезка** - создает скриншот с точной обрезкой до границ таблиц
4. **Отправка** - отправляет скриншот в Telegram чат с подписью

## Устранение неполадок

### Ошибка авторизации
- Проверьте учетные данные в `table_screenshot.py`
- Убедитесь, что сайт callcenter.hatimaki.ru доступен

### Ошибка создания скриншота
- Проверьте, что playwright установлен: `playwright install chromium`
- Убедитесь, что браузер может запуститься в headless режиме

### Ошибка отправки в Telegram
- Проверьте токен бота и ID чата в `.env`
- Убедитесь, что бот имеет права на отправку сообщений в чат
